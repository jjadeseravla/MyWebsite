# My Website

## Tech Used
- JavaScript
- Node.js with Express
- Mocha and Chai
- Mongo with Mongoose
- CSS and SASS   
- HTML  
- Docker

## To Run Application
1 Clone this repo and navigate to it.
2 ```npm install```
3 ```nodemon```
4 Start server on ```localhost:3000```

### To Run Tests
```npm install mocha -save ```
1 ```npm test```

## Styling Changes
```npm run sass```
to copy sass to css
